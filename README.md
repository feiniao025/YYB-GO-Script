# YYB_GO

YYB Go 适配版。

## 环境变量

| 变量名 | 格式 | 示例 |
|--------|------|------|
| `YYB_GO` | `地址@openid`，支持换行或 `&` 分隔多个账号 | `172.17.0.4:8000@openid1&172.17.0.4:8000@openid2` |

## 青龙订阅

```
ql repo https://github.com/feiniao025/YYB-GO-Script.git "" "" "" ""
```

## 注意事项

- 不要带 `http://` 前缀
- 不要用容器名（如 `yyb-go`），用 IP
- 脚本不会同时执行，青龙定时任务需手动错开 cron
